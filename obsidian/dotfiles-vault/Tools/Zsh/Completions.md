# Zsh — Completion System

> How tab-completion works, what's auto-generated, and how to add more.

## Architecture

```
~/.config/zsh/completions/   ← custom + generated completion files
    _gh                      ← generated by: gh completion -s zsh
    _pnpm                    ← generated by: pnpm completion zsh
    _mise                    ← generated by: mise completion zsh
    _kubectl                 ← generated by: kubectl completion zsh
    _rustup / _cargo         ← generated by: rustup completions zsh [cargo]
    _wt                      ← generated by: wt completion zsh
    _uv                      ← generated by: uv generate-shell-completion zsh
~/.cache/zsh/
    zcompdump                ← compiled completion dump (refreshed daily)
    zcompcache/              ← per-tool completion cache
```

## Lazy generator

The `_zsh_gen_completion` helper only regenerates a file when the tool binary is **newer** than the cached file:

```zsh
_zsh_gen_completion gh ~/.config/zsh/completions/_gh gh completion -s zsh
```

This means: update `gh` → next shell start regenerates `_gh`. No manual step needed.

## `compinit` caching

```zsh
# Regenerate at most once per day
if [[ -n "$HOME/.cache/zsh/zcompdump"(#qN.mh+24) ]]; then
  compinit -d "$HOME/.cache/zsh/zcompdump"    # full rebuild
else
  compinit -C -d "$HOME/.cache/zsh/zcompdump" # use cache (-C)
fi
```

This saves ~40ms on startup.

## `zstyle` settings

| Style | Value | Effect |
|-------|-------|--------|
| `file-list` | `all` | Always show file details in completion |
| `menu` | `select` | Arrow-key navigation in completion menu |
| `use-cache` | `on` | Cache completions per tool |
| `matcher-list` | `m:{a-z}={A-Z}` | Case-insensitive matching |

## fzf-tab

The `Aloxaf/fzf-tab` plugin replaces the menu with a floating fzf picker. Press `<Tab>` to get a fuzzy-searchable list instead of the default cycling menu.

## Adding a new tool's completions

If a tool supports `<cmd> completion zsh`, add a line to `.zshrc`:

```zsh
_zsh_gen_completion mytool "$HOME/.config/zsh/completions/_mytool" mytool completion zsh
```

The `_*` files are gitignored — they're generated at runtime.
